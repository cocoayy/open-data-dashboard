# Open Data Dashboard

オープンデータを活用し、需要傾向の可視化・比較・相関分析を行うデータ分析ダッシュボードです。

## ⚠️ 現在の開発状況と課題

本プロジェクトは現在、開発途中の段階です。方向性を変えるため、終了いたしました。  
CSVファイルをアップロードし、ダッシュボードを生成する機能まで実装しましたが、  
現時点ではいくつかの課題が残っています。

---

## 🧩 現在の実装機能

- CSVアップロード機能  
- 列マッピング機能（任意の列名に対応）  
- データプレビュー表示  
- フィルタ機能（期間指定）  
- KPI表示（平均・最大・最小）  
- 可視化  
  - 時系列グラフ  
  - 棒グラフ  
  - 散布図（相関分析）  
- 相関係数の算出と簡易説明  

---

## 🚧 課題

現状の実装には以下の課題があります。

- データ構造を「年月・販売量・気温」に固定しており、汎用性が低い  
- CSVのフォーマットに強く依存している  
- 分析内容が限定的で、ユーザーごとのニーズに対応できていない  
- 「ダッシュボード生成ツール」としては柔軟性が不足している  
- データの意味（ドメイン）に依存した設計になっている  

---

## 🔄 方針転換について

当初は「特定のデータ（ガス販売量など）を分析するダッシュボード」を目指していましたが、  
開発を進める中で、

**より汎用的にデータを分析できるツールの方が価値が高い**

と判断しました。

そのため、現在の実装をベースにしつつ、  
プロダクトの方向性を見直すことにしました。

---

## 🚀 今後の方向性

今後は以下の方向に設計を拡張・改善していきます。

### ① 汎用データ分析ツールへの拡張

- 任意のCSVに対応  
- 列の意味を動的に解釈  
- 数値列・カテゴリ列の自動判定  

---

### ② 自動可視化機能

- 列の組み合わせから自動でグラフを提案  
- 分布・比較・相関などを自動生成  

---

### ③ 分析支援機能の強化

- 基本統計量（平均・分散など）の自動算出  
- 異常値検知  
- 簡易レポート生成  

---

### ④ UXの改善

- より直感的な操作フロー  
- データ読み込み〜分析までの一貫した体験  

---

## 🧠 学び

本プロジェクトを通じて、以下の点を学びました。

- データ分析ツールにおける「汎用性」の重要性  
- 技術実装だけでなく「誰の課題を解決するか」を考える重要性  
- 実際に使われるプロダクトには柔軟性が不可欠であること  
- 開発を進めながら設計を見直すことの大切さ  

---

## 📌 補足

本プロジェクトは、オープンデータ可視化評価のプロダクトに移行いたしました。
