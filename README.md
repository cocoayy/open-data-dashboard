# Energy Data Dashboard

オープンデータを活用し、需要傾向の可視化・比較・相関分析を行うデータ分析ダッシュボードです。

## 背景
企業や自治体ではデータが存在していても、前処理や可視化に時間がかかり、十分に活用されていないケースがあります。
本プロジェクトでは、データの整形から可視化、分析までを一貫して行い、意思決定を支援することを目的とします。

## 想定ユーザー
- データ活用を推進したい企業の現場部門
- エネルギー需要や販売傾向を分析したい担当者
- オープンデータを用いて分析を行いたい学生・初学者

## 主な機能（予定）
- データの前処理
- 時系列可視化
- 地域別比較
- 相関分析
- 分析結果の考察表示

## 技術スタック（予定）
